# Salesforce Documentation Generator

Generate comprehensive Salesforce documentation including job aids, technical specs, runbooks, and user guides.

## What It Generates

- **Job Aids**: User-friendly, step-by-step guides with screenshots
- **Technical Specs**: Detailed architecture, data model, integration docs
- **Runbooks**: Deployment and troubleshooting procedures
- **User Guides**: Comprehensive feature documentation
- **Release Notes**: What's new, what changed

## Installation

```bash
cp -r SalesforceDocumentation $PAI_DIR/Skills/
```

## Usage

```
"Create a job aid for sales reps on territory lead assignment"
"Generate technical documentation for the enterprise routing system"
"Create a runbook for deploying the PDF generation feature"
```

## Workflows

- **JobAid.md** - End-user documentation
- **TechnicalSpec.md** - Developer/architect documentation
- **Runbook.md** - Operational procedures

## Benefits

- Consistent documentation format
- All edge cases covered
- Screenshots placeholders included
- Troubleshooting sections
- Ready for real-world use
