# Salesforce Data Migration Generator

Generate data migration plans, field mappings, transformation scripts, and bulk API code.

## What It Generates

- **Field Mappings**: Detailed source-to-target mappings
- **Migration Plans**: Phased rollout strategies
- **Bulk API Scripts**: Load 50,000+ records efficiently
- **Data Transformations**: Cleanup and format conversion logic

## Installation

```bash
cp -r SalesforceDataMigration $PAI_DIR/Skills/
```

## Usage

```
"Create field mapping from legacy CRM to Salesforce Account"
"Generate migration plan for 100,000 contacts"
"Build bulk API script for opportunity data load"
```

## What You Get

- ✅ Complete field mappings with transformations
- ✅ Edge case handling (nulls, duplicates, invalid data)
- ✅ Validation rules impact analysis
- ✅ Rollback procedures
- ✅ Testing plan
- ✅ Sign-off checklist
