# Salesforce Testing Generator

Generate comprehensive test classes, test data builders, and test scenarios.

## What It Generates

- **Apex Test Classes**: Complete with @testSetup, bulk testing, negative cases
- **Test Data Builders**: Reusable builder pattern classes
- **LWC Tests**: Jest tests for Lightning Web Components

## Installation

```bash
cp -r SalesforceTesting $PAI_DIR/Skills/
```

## Usage

```
"Generate test class for OpportunityTriggerHandler"
"Create a test data builder for Account and Opportunity"
"Write Jest tests for the opportunityProductList component"
```
